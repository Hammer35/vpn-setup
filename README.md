# VPN-сервер: установка с нуля

Этот скрипт автоматически поднимает VPN-сервер на базе **3x-ui + XRay (VLESS + Reality)** на чистом сервере Ubuntu.

Протокол **VLESS + Reality** маскирует трафик под обычный HTTPS — его сложнее заблокировать, чем стандартный VPN.

---

## Что тебе нужно

| # | Что | Где взять |
|---|-----|-----------|
| 1 | VPS с Ubuntu 22.04 или 24.04 | Hetzner, DigitalOcean, Timeweb, Aeza и др. |
| 2 | SSH-клиент на Windows | Уже встроен в Windows 10/11 |
| 3 | 5–10 минут времени | — |

**Минимальные требования к серверу:** 1 CPU, 512 MB RAM, любой тариф. Порты 443 и 2053 должны быть открыты.

---

## Шаг 1 — Открой PowerShell

На Windows нажми **Win + X** → выбери **"Терминал"** или **"Windows PowerShell"**.

---

## Шаг 2 — Подключись к серверу

В PowerShell введи:

```
ssh root@IP_ТВОЕГО_СЕРВЕРА
```

Замени `IP_ТВОЕГО_СЕРВЕРА` на реальный IP, который дал хостер.

Пример:
```
ssh root@185.100.200.50
```

Если спросит **"Are you sure you want to continue connecting?"** — напечатай `yes` и нажми Enter.

Введи пароль от сервера (его даёт хостер при создании).

---

## Шаг 3 — Запусти установку

Когда подключился (в строке появится `root@...#`), введи одну команду:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Hammer35/vpn-setup/main/install.sh)
```

Нажми Enter и жди. Скрипт сам всё сделает.

---

## Что происходит во время установки

Скрипт проходит 6 этапов — ты видишь прогресс в реальном времени:

```
[•] ЭТАП 1/6 — Обновление системы
[•] ЭТАП 2/6 — Установка панели 3x-ui
[•] ЭТАП 3/6 — Генерация ключей Reality
[•] ЭТАП 4/6 — SSL сертификат
[•] ЭТАП 5/6 — Настройка панели и VPN
[•] ЭТАП 6/6 — Запуск сервера
```

Занимает около **5–10 минут**.

## Шаг 4 — Сохрани данные

После завершения скрипт покажет:

```
================================================
       ВСЁ ГОТОВО! СОХРАНИ ДАННЫЕ!
================================================

  Панель управления VPN:
  https://185.100.200.50:2053/abc123def456/

  Логин:  admin
  Пароль: XyZ9qRmT2w

  VPN ссылка — скопируй в v2rayN / Hiddify:
  vless://uuid@ip:443?...#VPN-185.100.200.50
================================================
  ВАЖНО: Сохрани эти данные сейчас!
```

**Скопируй и сохрани всё это** — пароль и VPN-ссылка генерируются один раз и нигде не хранятся.

## Шаг 5 — Подключись к VPN

### Скачай клиент (на Windows):

**v2rayN** — рекомендуется:
- Скачай последний релиз: [github.com/2dust/v2rayN/releases](https://github.com/2dust/v2rayN/releases)
- Файл: `v2rayN-windows-64.zip`

**Hiddify** — проще для новичков:
- Скачай: [github.com/hiddify/hiddify-app/releases](https://github.com/hiddify/hiddify-app/releases)

### Импортируй VPN-ссылку:

В **v2rayN**: нажми `+` → "Импорт из буфера обмена" → вставь ссылку `vless://...`

В **Hiddify**: нажми `+` → вставь ссылку `vless://...`

## Панель управления

Открой в браузере адрес панели (из вывода скрипта):

```
https://IP:2053/твой_секретный_путь/
```

Браузер может показать предупреждение о сертификате — нажми **"Продолжить"** (это нормально для IP-адреса).

В панели можно:
- Добавлять новых пользователей
- Смотреть статистику трафика
- Управлять подключениями

## Частые вопросы

**Q: Скрипт завис на "Установка 3x-ui"**
A: Подожди — это нормально, загрузка может занять 2–3 минуты.

**Q: Браузер не открывает панель**
A: Убедись что порт 2053 открыт у хостера (Firewall / Security Groups).

**Q: SSL сертификат не выпустился**
A: Скрипт создаст самоподписанный — панель будет работать, просто браузер покажет предупреждение.

**Q: Забыл пароль от панели**
A: Зайди на сервер и выполни: `x-ui reset-user`

## Требования к серверу

- ОС: **Ubuntu 22.04** или **Ubuntu 24.04** (другие не поддерживаются)
- Права: **root**
- Открытые порты: **443** (VPN) и **2053** (панель)
- ОЗУ: от 512 MB
