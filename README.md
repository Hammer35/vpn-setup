# Как поднять свой VPN с нуля

Этот скрипт сам установит всё необходимое на твой сервер. Тебе нужно только скопировать и вставить несколько команд. Никакого программирования — просто следуй шагам.

---

## Что тебе нужно перед началом

**1. Сервер (VPS)**
Это компьютер в интернете, на котором будет работать VPN. Его нужно арендовать.

Где арендовать (выбери любой):
- [Hetzner](https://www.hetzner.com) — надёжный, дешёвый (от €4/мес)
- [Aeza](https://aeza.net) — принимает рублями
- [DigitalOcean](https://www.digitalocean.com) — популярный

При создании сервера выбери:
- ОС: **Ubuntu 24.04**
- Тариф: самый дешёвый (1 CPU, 1 GB RAM)

После создания хостер даст тебе **IP-адрес** и **пароль** — сохрани их.

---

**2. Терминал на Windows**
Он уже есть на твоём компьютере. Ничего устанавливать не нужно.

Как открыть: нажми **Win + R**, в появившемся окне напечатай `powershell` и нажми **Enter**

---

## Установка — 4 простых шага

### Шаг 1. Открой терминал

Нажми **Win + R**, напечатай `powershell`, нажми **Enter**

Откроется чёрное (или синее) окно. Это нормально.

---

### Шаг 2. Подключись к своему серверу

Введи эту команду (замени `185.100.200.50` на твой IP):

```
ssh root@185.100.200.50
```

Нажми **Enter**.

Если появится вопрос:
```
Are you sure you want to continue connecting (yes/no)?
```
Напечатай `yes` и нажми Enter.

Затем введи пароль от сервера (его дал хостер). Когда вводишь пароль — символы не отображаются, это нормально. Просто введи и нажми Enter.

Когда подключился, строка будет выглядеть так:
```
root@ubuntu:~#
```

---

### Шаг 3. Запусти установку

Скопируй эту команду полностью и вставь в терминал (правая кнопка мыши = вставить):

```
bash <(curl -Ls https://raw.githubusercontent.com/Hammer35/vpn-setup/main/install.sh)
```

Нажми **Enter**.

Установка займёт **5–10 минут**. Ты будешь видеть что происходит:
```
[•] ЭТАП 1/6 — Обновление системы
[•] ЭТАП 2/6 — Установка панели 3x-ui
[•] ЭТАП 3/6 — Генерация ключей
...
```

Просто жди, ничего не нажимай.

---

### Шаг 4. Сохрани данные

Когда всё установится, в конце появится:

```
================================================
       ВСЁ ГОТОВО! СОХРАНИ ДАННЫЕ!
================================================

  Панель управления VPN:
  https://185.100.200.50:2053/abc123def456/

  Логин:  admin
  Пароль: XyZ9qRmT2w

  VPN ссылка — скопируй в v2rayN / Hiddify:
  vless://......#VPN-185.100.200.50

================================================
  ВАЖНО: Сохрани эти данные сейчас!
================================================
```

**Скопируй всё это в блокнот и сохрани.** Пароль и VPN-ссылка генерируются один раз.

## Подключение к VPN на Windows

### Скачай программу v2rayN

Это бесплатная программа для подключения к VPN.

1. Перейди по ссылке: [github.com/2dust/v2rayN/releases](https://github.com/2dust/v2rayN/releases)
2. Найди последний релиз вверху
3. Скачай файл с названием **`v2rayN-windows-64.zip`**
4. Распакуй архив в любую папку
5. Запусти **`v2rayN.exe`**

### Добавь VPN-соединение

1. В программе v2rayN нажми кнопку **"+"** (или Ctrl+V)
2. Выбери **"Импорт из буфера обмена"**
3. Вставь VPN-ссылку `vless://...` которую сохранил
4. Нажми **"Подключить"**

Готово — VPN работает.

## Если что-то пошло не так

**Не могу подключиться к серверу по SSH**
→ Проверь IP-адрес — он должен совпадать с тем, что дал хостер.
→ Убедись что сервер запущен (в личном кабинете хостера).

**Установка зависла и ничего не происходит 10+ минут**
→ Нажми **Enter** один раз — иногда терминал просто ждёт подтверждения.

**Браузер пишет "Небезопасно" при открытии панели**
→ Нажми **"Дополнительно"** → **"Перейти на сайт"**. Это нормально для IP-адреса.

**Забыл пароль от панели**
→ Зайди на сервер по SSH и введи: `x-ui reset-user`

**Панель не открывается в браузере**
→ В личном кабинете хостера найди раздел **Firewall** и открой порты **443** и **2053**.
